# Trilha-Do-Eleitor

A plataforma Trilha do Eleitor (TDE) é um sistema web educacional desenvolvido com o objetivo de promover a educação eleitoral de forma acessível, interativa e progressiva. A plataforma oferece trilhas de aprendizagem compostas por conteúdos em texto, vídeos e quizzes, permitindo que os usuários desenvolvam conhecimento sobre o processo eleitoral brasileiro.
Além disso, a plataforma disponibiliza um simulador de votação, que permite ao usuário praticar o uso de uma urna eletrônica em um ambiente seguro e educativo, reforçando o aprendizado prático.
O sistema também contempla diferentes perfis de usuários, como usuários comuns, conteudistas e administradores, possibilitando a criação, organização e gerenciamento de conteúdos educacionais de forma estruturada.
